Decalaration And Initialization of Array using Literal

Using Array Literal 
Syntax:- var array_name = [];   //empty array

Example:

        var geek = [];      //Declaration

        //Initialization

        geek[0] = "Rahul";
        geek[1] = "Mohit";
        geek[2] = "Rohit";
        geek[42] = "Mohan";  //It is recommened that write array Initialization in a sequence index no:

It is recommened that write array Initialization in a sequence index no:

Syntax: Declaration and Initialization together

        var array_name = [value1 , value2, value_n];

Example:-

        var geek1 = ["Rahul","Mohit","Rohit",12,"Jay"];

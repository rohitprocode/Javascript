Return Statement

A Return Statement may return any type of data, including arrays and objects. The return statements stop the execution of a function and returns a value.

Syntax: 1

return( Variable or expression );

Example: 1

return(3); return(a+b); return(a);

Syntax : 2

function function_name(para1 , para2 , ....) { block of statements / body of code; return( variable / expression / array / object ); } function_name();

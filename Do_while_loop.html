Do While Loop

The do while loop is similar to while loop, but the condition is checked after the loop body is executed.This is ensure that the body is run at least once.

Syntax: 1

Initialization; do { body of code/block of statements; Increment/Decrement; } while( Test Condition);

Example:

Syntax: 2

Initialization; do { if(Test Condition) { body of code / block of statements; Increment / Decrement; } } while( true / false);

Tagged Template

Tagged templates are advance form of template literal tag allow you to parse template literals with a function.The first argument of a tag function contains an array of string values.The remaining arguments are related to the expressions.
In the end,your function can return your manipulated string.

Example:

If there are 5 movie tickets each rs. 200 and if you buy 5 tickets get discount Rs.50 each.
There are 5 movie tickets each Rs.200 and if you buy less than 5 tickets get discount Rs. 0 each.

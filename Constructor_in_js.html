Constructor in Javascript

Object instance are created with Constructor,which are basically special function that prepare new instance of an object for use.

Syntax:-

        function Constructor_name(){
                        this.key_name = key value;
                        this.key_name2 = key value2;
                        //you can write more properties like this.
                        this.method = function() {
                            block of statements;
                        }
        }

        //Now you can declare Constructor with object name.

            var object_name = new Constructor_name();
        
        Now, You can access the properties like:

            object_name.key_name/property_name;
        
        or, You can access the method like:-
            object_name.method_name();

Example:

Constructor With Parameter

Syntax: 2

function function_name(parameter1,para2,para3....etc){
                this.key_name = key value/parameter1;
                this.key_name2 = key value/parameter2;
                //many more like this.
                this.method = function(){
                            block of statements;
                }
        }
        
        var object_name = Constructor_name(); //pass value in it like: 
        var object_name = Constructor_name(para1,para2);
        //You can also call it like:
        Object_name.key_name;
        //or call method like:
        Object_name.method();

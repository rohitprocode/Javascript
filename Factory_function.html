Factory Function in Javascript

When a function returns an object,we call it a factory function.It can produce object instance without new Keyword or classes.

Syntax: //You can create it with or without parameters.

    function function_name(parameter1,parameter2 and many more){
        return { 
            key1: parameter1,
            key2: parameter2,
            //we can write many more keys, parameter and parameter values if we need.
            key3: function(){ block of statements };
        }
    }
    //How to call it:-
    var object_name = function_name( //you may pass parameter value in it.);
    //You can declare and call many different objects for use of this function(Factory Function).

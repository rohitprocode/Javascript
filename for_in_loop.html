for in loop in Javascript

The for in loop goes all the properties of object and iterate it one by one.The for in loop is used to loop through an objects properties.
The for in loop in Javascript allows you to iterate over all property keys of an object.

Syntax:1 

 for (var variable_name in Object_name){
        block of statements;
    }

Example : 1

        In the place of variable_name we can write any word which is suitable for our mind.
        for(var abc in MOBILE){ 
            document.write(abc);
        }

Example:2

We notice that METHOD also displayed in the output


We saw in previous exmaple that function (method) also displayed in for in loop statement.So, if we want to only display object properties without method then we can write code like:-

Syntax:-2

    for(var variable_name in Object_name){
        if(typeof Object_name[variable_name] !== 'function'){
            block of statements;
        }
    }

Example:-3

In the Previous Example we get only values.If you want to display properties name also you can write code like:-

Syntax:-

    for(var variable_name in Object_name){
        if(typeof Object_name[variable_name] !== 'function'){
            block of statements;
        }
    }
    //We write special code in block of statements;

We have one more way to iterate / display object properties:-
The Object.keys() feature returns an array of a given objects own enumerable property names.

syntax:-

    Object.keys(Object_name);   //Object's O is Capital.

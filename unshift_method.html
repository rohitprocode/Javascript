unshift() method

The unshift() method adds one or more elements to the beginning of an array and returns the new length of the array.This method changes the length of an array.

Syntax: 

            Array_name.unshift(value1,value2,value_n);

Syntax for length

            var variable_name = Array_name.unshift(value1,value2,value_n);

            document.write(variable_name);

            //It will return length of the array/new length of the array

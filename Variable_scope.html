Variable Scope

Global Scope / Global Variable

A variable that is declared outside a function definition is a global variable, and its value is accessible and modifiable throught your program.

In a web browser, Global variables are deleted when you close the browser window ( or tab ), but remain available to new pages loaded into the same window.

//If we initialize a value outside a function then it is called as global variable.
//If we intialize a value inside a function then it is called as local variable.
//But, If we initialize a value inside a function a function without " var " then it is also called as global variable.

Syntax : 1

        Initialization;
        function function_name(){
            block of statements ;               //Global Variable
        }
        function_name();

Local Scope / Local Variable

A variable that is decalared inside a function definition is local.It is created and destroyed everytime the function is executed, and it cannot be accessed by any code outside the function.Inside a function, if a variable has not been declared with " var " it is created / treated as a global variable.

Syntax: 1

        function function_name(){
            Initialization;         //Initialize using var   //Local variable
            block of statements;
        }
        function_name();

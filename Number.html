Number

Number type in javascript includes both integar and floating point values.Javascript numbers are always stored as double precision floating point numbers, following the international IEEE 754 standard.
Javascript also provides an object representation of numbers.

Example:

            1.12
            23.45
            5e3

            1.meaning of 5e3 = 5*10^3 = 5 x 10 x 10 x 10 = 5000
            2. meaning of 234e-5 = 0.00234

Identifier

An identifier ia a name having a few letters, numbers and special characters, _ (underscore). It is used to identify a variable, function, symbolic constant and so on.

Example:- const a = "hello", int student, float marks. (a,student,marks is a identifier). You should know: 1.Identifier names cannot start with numbers. 2.Javascript is case-sensitive. 3.Keywords cannot be used as identifier names.

Function in Javascript

Function are subprograms which are used to compute a value or perform a a task.

According to W3 School:

A Javascript Function is a block of code designed to perform a particular task.

A Javascript Function is executed when "Something" invokes it (calls it).

Types of Function:-

-Library or Built in Function

Javascript libraries is a file that contains a set of prewritten function or codes that you can repeatedly use while executing javascript tasks. Developers may plug in javascript library codes as needed to your code.

Example:

valueOf() , write(), alert() etc.

-User-defined functions

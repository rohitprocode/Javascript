Template Literal / Template String

Template Literals are string literals allowing embedded expression.You can use multiline strings and string interpolation features with them.
Template literals are enclosed by the back-tick(``) character instead of double or single quotes.

double quotes "" or single quotes '' are same but back-tick `` is different between them.

Multiline String:

Example:

            //It will show error,
            var str = "Hello
                       Rohit";

            but, 

            //It works without any error
            var str = `Hello
                       Rohit`;

String Interpolation

Template literals can contain placeholders.These are indicated by the dollar sign and curly braces (${expression})

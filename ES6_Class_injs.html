ES6 Class in Javascript

Javascript Classes, introduced in ECMAScript 2015 or ES6,Classes are in fact "Special Function".

There are Two ways to define class in Javascript using class keyword:

1. Class Declaration
2. Class Expression 
                            //Class Declaration
            
            
Syntax:


            class class_name{
                constructor(){
                    Properties/Methods          //Instance Member
                }
                Methods                         //Prototype Member
            }

Constructor

The Constructor method is a special method for creating and initializing an object created within a class. There can be only one special method with the name "Constructor" in a class. 
            
Syntax:

            
            class class_name{
                constructor(){
                    properties/Methods
                }
                Methods

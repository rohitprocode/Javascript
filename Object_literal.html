Declaration and Initialization of Object

Using Object Literal:

JavaScript Object Literal is a data type used to define objects in JavaScript. It is a syntax for creating an object in JavaScript that is composed of key-value pairs. It is a lightweight and efficient way to create and store data. It is a great way to store related data in an organized manner.

Syntax:

 var object_name = { };

Example :

        var fees = { };

How to Write properties of object in two different ways:

Example:

        //Square Bracket 

        fees ['Rahul'] = 100;
        fees ['Sumit'] = 200;
        fees ['Mohan'] = 300;

        // " " is required in multiword.
        fees ["Super Man"] = 400;

        //Dot .

        fees.Rahul = 100;
        fees.Sumit = 200;
        fees.Mohan = 300;
        
        // . is not applicable in dot feature you have to use Square Brackets
        fees["Super Man"] = 400;

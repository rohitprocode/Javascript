Set Methods:

            //SetMethods
            Method	            Description
            setFullYear()	    set year as a four digit number (yyyy)
            setMonth()	        set month as a number (0-11)
            setDate()	        set day as a number (1-31)
            setDay()	        set weekday as a number (0-6)
            setHours()	        set hour (0-23)
            setMinutes()	    set minute (0-59)
            setSeconds()	    set second (0-59)
            setMilliseconds()	set millisecond (0-999)
            setTime()	        set time (milliseconds since January 1, 1970)

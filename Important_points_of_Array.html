Important Points about Array

  • Javascript arrays are zero-indexed: The first element of an array is at index 0
  • Using an invalid index number returns undefined
  • It's possible to quote the javascript array indexes as web (eg. geek['2] instead of geek[2]), although its not necessary.
  • Arrays can not use strings as element indexes but must use integers.
  • There is no associative array in javascript:-
    geek["fees"] = 200;
  • No advantage to use array constructor so better to use Array literal for creating Arrays in javascript.

String

String is built-in object corresponding to the primitive string data type. String is a group of characters.

Example:

//You can write string with double quotes or single quotes or Backtick like:
"Welcome" or 'Welcome' or `Welcome`
"Geeky" or 'Geeky' or `Backtick`
"12345" or '12345' or `12345`

//You can use any of them all are correct.

Example : for display

Example: How to store variable in string

Primitive: type is string

  • var str = "Hello";
  • var str = 'Hello';
  • var str = `Hello`;

Constructor: type is object

  • var str = new String("Hello");
  • var str = new String('Hello');
  • var str = new String(`Hello`);

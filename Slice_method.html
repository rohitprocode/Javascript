slice() method

The slice() method returns a shallow copy of a portion of an array into a new array object selected from begin to end (end not included).The original array will not be modified.

Syntax:

            array_name.slice(begin_value,end_value);
Start (begin_value)

  • If begin is undefined, slice begins from index 0.

  • If begin is greater than the length of the sequence, an empty array is returned.

  • A negative index can be used, indicating an effect from the end of the sequence.slice(-2) extracts the last two elements in the sequence.

End (end_value)

  • The end is omitted,slice exracts through the end of the sequence (arr.length);

  • If end is greater than the length of the sequence,slice extracts through to the end of the sequence(arr.length).

  • A negative index can be used,indicating an effect from the end of the sequence.slice(2-1) extracts the third element through the second to last element in the sequence.

  • slice extracts up to but not including end.

QuerySelectorAll("CSS Selector")

The method QuerySelectorAll() returns a static (not live) NodeList representing a list of document's elements that the specified group of selectors.

  • The NodeList objects represents a collection of nodes.The nodes can be accessed by index numbers.The index start at 0.
  • You can use the length property of the NodeList object to determine the number of elements that matches the specified selector,then you can loop through all elements and extract the info you want.
  • If the specified selector include a CSS pseudu-element the returned list is always empty.

We Can target Element,ID,Class and Attribute with QuerySelectorAll()

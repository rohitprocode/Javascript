Rest Parameter

The Rest parameter allows to represent an infinite number of arguments as an array.

The Rest operator must be the last parameter to a function.

Syntax: 1

function function_name(...args) { block of statements / body of code }

Syntax: 2

function function_name(a , ...args) { block of statements / body of code }

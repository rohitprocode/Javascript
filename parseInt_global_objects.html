parseInt()

The parseInt function parses a string and returns and integer.

Syntax:

            parseInt(string,radix);

The radix parameter is used to specify which numeral system to be used, for example a radix of 16(hexadecimal) indiacates that the number in the string should be parsed from a hexadecimal number to a decimal number.
If the redix parameter is omitted,javascript assumes the following:
1. If the string begin with "0X",the radix is 16(hexadecimal).
2. If the string begins with any other value, the radix is 10(decimal).

Only the first number in the string is returned.
Leading and trailing spaces are allowed.
If the first character cannot be converted to a number, parseInt() returns NaN.

Creating Date Object

• new Date() - new Date creates a new date object with the current date and time.

Example:

• new Date(milliseconds) - It creates a new date object as January 1, 1970, 00:00:00 Universal Time (UTC).

Example:

• new Date(year,month,day,hours,minutes,seconds,milliseconds) = It creates object with the date specified by the integer values for the year,month,day,hours,minutes,seconds,milliseconds.You can omit some of the arguments. 

            With 7 arguments : Year, Month, Day, Hour, Minute, Second, millisecond 
            With 6 arguments : Year, Month, Day, Hour, Minute, Second
            With 5 arguments : Year, Month, Day, Hour, Minute
            With 4 arguments : Year, Month, Day, Hour
            With 3 arguments : Year, Month, Day
            With 2 arguments : Year, Month
            With 1 arguments : milliseconds
        
            Month and Week day start with 0.
            0 - Sunday
            0 - January
            Month Day start with 1
               1 - 1

Example:

• new Date(dateString) - new Date(dateString) creates a new date object from a date string.

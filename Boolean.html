Boolean

Boolean is the built in object corresponding to the primitive Boolean data type.Javascript boolean can have one of two values: true or false

primitive values

  • var primitiveTrue = true; //primitiveTrue is a variable name.
  • var primitiveFalse = false; //primitiveFalse is a variable name.

Boolean Function

  • var functionTrue = Boolean(true); //functionTrue is a variable name.
  • var functionFalse = Boolean(False); //functioFalse is a variable name.

Boolen Constructor

  • var constructorTrue = new Boolean(true); //constructorTrue is a variable name.
  • var constructorFalse = new Boolean(false); //constructorFalse is a variable name.

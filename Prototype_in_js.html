Prototype in Javascript

A Prototype is an existing in built functionality in javascript.Whenever we create a javascript function, javascript adds a prototype property to that function.A prototype is an object, where it can add new variables and methods to the existing object.

Syntax:

           class_name.prototype.key = 'value';
    Example:    Mobile.prototype.color = "Green";
//This property of prototype is now applied on all objects which we declared on Mobile Class or Object.

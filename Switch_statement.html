Switch statement in js

Switch statement check several possible values for an expression.

Syntax:

switch(expression){ case 1: block of statement; break; case 2: block of statement; break; case 3: block of statement; break; default: block of statement; }

If you want to get same output for different cases, then:

Syntax:

if(expression){ case 1 : block of statement; break; case 2 : block of statement; break; case 3: case 4: block of statement; case 5: block of statement; break; default: block of statement; }

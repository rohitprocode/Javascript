Splice() Method

The splice() method changes the contents of an array by removing existing elements and/or adding new elements.These method changes the original array.

Syntax:

            var array_name.splice(start_value,deleteCount_value,replace_value)

  • Start:-

    The first argument start specifies at what position to add/remove items, use negative values to specify the position from the end of the array.

  • deleteCount_value

    The second argument deleteCount, is the number of elements to delete beginning with index start.

Default Parameter

The default parameter is a way to set default values for function parameters a value is no passed in.(ie. it is undefined)

In a function, If a parameter is not provided, then its value becomes undefined. In this case the default value that we specify is applied by the compiler.

Syntax : 1

function function_name( para1 , para2 , para3 = "value"){ block of statements / block of code }

Syntax : 2 // This Problem is undefined

function function_name( para1 , para2 = "Value" , para3) { block of statements/block of code }

The default value can be written only at the end of the parameter.

Static Method in Class

The Static method is used to define a static method for a class.Static methods are called without creating object and cannot be called through a class instance (object).Static methods are often used to create utility functions for an application.
Static Means Class Level.

Syntax:

            class class_name{
                constructor(){
                    properties/methods;
                }
                static method;
            }

            class_name.static_method_name();

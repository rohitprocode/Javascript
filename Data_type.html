Data Type

In Javascript we do not need to specify type of the variable because it is dynamically used be javascript engine. We can use var data type. It can hold any type of data like string, Numbers, Boolean etc.

    Primitive Data Type
  1. string
  2. Number
  3. Boolean
  4. Undefined
  5. Null
    Non-Primitive Data Type
  1. Object
  2. Array
  3. RegExp

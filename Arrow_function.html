Arrow Function in Javascript

An Arrow function expression (previously and now incorrectly known as fat arrow function) has a shorter syntax as compared to function expression.Arrow functions are always anonymous function.
//Shorter
//Always anonymous

Syntax:-

        () => { block of statements / body of code };

        //Comparison with another function              

        //Another Function (Function Expression)
        var myfun = function show(){
            document.write("Rohit Shows");
        }
        myfun();          

        //Arrow Function

        var myfun = () => {document.write("Rohit Shows");};
        myfun();

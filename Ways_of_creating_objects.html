Ways of Creatng Objects in Javscript

If you have done your work with single object then you should use:-
Object Literal or Object Constructor
If you wants to make many objects with the property then you should use Factory function or Objects Constructor function.

Object Literal

        //Declare and Initialize Separately

        var fees = {};
        fees['Rahul'] = 200;

        //Object literal Declare and Initialize together

        var fees = { Rahul : 200 };

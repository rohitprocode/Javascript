Ways of Creatng Objects in Javscript

If you have done your work with single object then you should use:-
Object Literal or Object Constructor
If you wants to make many objects with the property then you should use Factory function or Objects Constructor function.

Object Literal

        //Declare and Initialize Separately

        var fees = {};
        fees['Rahul'] = 200;

        //Object literal Declare and Initialize together

        var fees = { Rahul : 200 };

Object Constructor

        var fees = new Object();

Factory Function

        function mobile (){
            return{
                Model : 'Galaxy';
            }
        }
        
        //You can make Objects using this function

        var samsung = mobile();
        var Realme = mobile();

Constructor Function

        function Mobile(){
            this.Model = 'Galaxy';
        }

        //You can make objects using this Constructor

        var samsung = new Mobile();
        var Realme = new Mobile();
When you assign a Constructor function in a variable,It is called as Class.

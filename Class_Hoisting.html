Class Hoisting

Class Declaration and Class Expression are not be Hoisted.You first need to declare your class and then access it.

Right Examle:

        class Mobile{
            constructor(){
                Properties/Methods;
            }
            Methods;
        }

        var nokia = new Mobile();

Wrong Example:

        var nokia = new Mobile();

        class Mobile {
            constructor(){
                Properties/Methods;
            }
            Methods;
        }

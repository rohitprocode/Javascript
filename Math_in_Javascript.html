Math in Javascript

Math objects holds a set of constants and methods that enable more complex mathematical operations than the basic airthmetic operators.We can not instantiate Math object.The Math object is static so it's properties and methods accessed directly.

Example:

            Math.PI
            Math.abs()

Properties:

                Math.E        // returns Euler's number
                Math.PI       // returns PI
                Math.SQRT2    // returns the square root of 2
                Math.SQRT1_2  // returns the square root of 1/2
                Math.LN2      // returns the natural logarithm of 2
                Math.LN10     // returns the natural logarithm of 10
                Math.LOG2E    // returns base 2 logarithm of E
                Math.LOG10E   // returns base 10 logarithm of E

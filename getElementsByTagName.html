getElementsByTagName("Tag_Name")

The methods getElementsByTagName("tag_Name") returns a live node list meaning that it updates itself with the DOM tree automatically,so modification of the DOM tree will be reflected in the returned collection.The returned node list or Collection of nodes can be accessed by index numbers starting with index 0.

This method accepts the string indicating the type of elements that be retrieved,a special value "*" returns all elements in the documents.

You can use the length property of the NodeList Object to determine the number of elements with the specified tag name,then you can loop through all elements and extract the info you want.

Example:

getElementsByTagName("p")
// where p is name of tag

getElementsByTagName("*")
//The parameter value "*" returns all elements in the document.

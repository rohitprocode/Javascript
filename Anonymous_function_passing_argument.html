Anonymous Function Passing as Argument

Syntax: 1

        function function_name(parameter){
            return parameter_name();            //Remember, use () because we want to call Anonymous function here.
        }
        document.write(function_name(function(){
            return block of statement / body of code 
        }));

        Example:- 

         function disp(myfun){
             return myfun();                         //We use () because We get output from function (anonymous function)
        }
        document.write(disp(function(){
            return "Rohit Shows";
        }));

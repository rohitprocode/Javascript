Dom Event Model

The DOM 2 event model specification describes a standard way to create,capture,handle and cancel event in a tree like structure such as an XHTML document's object hierarchy.

Phases:

  • Capture Phase
  • Target Phase
  • Bubbling Phase

Capture Phase

The event goes down to the element.

Target Phase

The event reached the target element.

Bubbling Phase

The event bubbles up from the element.
